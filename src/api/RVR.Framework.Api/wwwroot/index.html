Démarrage rapide
  1. Vérifiez le statut d'initialisation du système
  2. Créez le premier admin via POST /api/init/first-admin
  3. Authentifiez-vous via POST /api/auth/login
  4. Utilisez le token JWT pour accéder aux endpoints protégés
Endpoints principaux
  • Initialization : /api/init/* Public
  • Authentication : /api/auth/* Public
  • Users : /api/users/*
  • Products : /api/products/*
